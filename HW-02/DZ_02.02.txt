                            Условия домашки
Домашнее задание на данном курсе состоит из двух блоков:
• Верстка сайта
• Задачи на кодинг на JS
Все домашние работы 2-го курса выполняйте в одном репозитории, назовите его 2-nd course hw.
Для верстки сайта в VS Code создайте файлы CSS и HTML.
Для выполнения задач по JS создайте файл hw-N, где N — номер урока на курсе.
Используйте уникальные имена переменных для каждой задачи, чтобы не было ошибок в работе программы.
После выполнения всех задач в домашнем задании сделайте Pull request и пришлите ссылку на него в качестве выполненного задания на платформу.
Не удаляйте и не закрывайте пул-реквест до тех пор, пока наставник не проверит вашу работу.
Если наставник не принял вашу работу или оставил в ней комментарии, не удаляйте и не закрывайте Pull request. Исправьте все недочеты и комментарии, сделайте commit и push. Pull request на GitHub обновится автоматически — в качестве исправленной работы присылайте ту же ссылку.
Если комментариев от наставника нет и задача выполнена на «Отлично»,  «влейте» ветку hw-N в ветку master с помощью merge.
При выполнении домашнего задания следующего урока продолжайте работу над версткой сайта в уже созданных файлах, а для решения домашнего задания по JS создайте новый файл hw N+1.
О том, как сдавать домашние задания через GitHub, вы можете прочитать в уроке «Основы Git и командной строки», в степе «Как сдать домашние задания через GitHub».

Задание 1
Создайте переменную a. Запишите в нее значение 10, выведите его на экран с помощью метода alert(). Затем запишите в нее значение 20, выведите его на экран.

Задание 2
Создайте переменную и присвойте ей значение года выпуска первого iPhone. Выведите значение переменной на экран.

Задание 3
Создайте переменную и присвойте ей значение имени создателя языка JavaScript. Выведите значение переменной на экран.

Задание 4
Создайте две переменные 10 и 2. С помощью метода alert() выведите на экран их сумму, разность, произведение и частное (результат деления).

Задание 5
Возведите 2 в 5-ю степень. Результат запишите в переменную result и выведите на экран с помощью метода alert().

Задание 6
Даны переменные: a = 9 и b = 2. Найдите остаток от деления a на b и выведите результат на экран с помощью метода alert().

Задание 7
Перепишите код с использованием операторов присваивания и инкремента/декремента (количество строк должно остаться прежним):
let num = 1;
num = num + 5;
num = num - 3;
num = num * 7;
num = num / 3;
num = num + 1;
num = num - 1;
alert(num);

Задание 8
Создайте переменную age, присвойте значение с помощью метода prompt() "Сколько вам лет?". Результат вывести с помощью alert().

Задание 9.0
Создайте объект user, у которого есть следующие свойства:  name, age, isAdmin. Значения для свойств придумайте произвольные. Обратите внимание, что для свойства name значение должно быть строковым, для age – числовым, а для isAdmin – булевым.

Задание 9.1
Добавьте объекту user свойство city of residence. Обратите внимание на то, как правильно добавлять такие методы: через точку или с помощью квадратных скобок.

Задание 9.2
Измените у объекта user свойство age на любое другое новое значение.

Задание 9.3
Удалите у объекта user свойство "city of residence".

Задание 9.4
Создайте переменную info, присвойте ей значение с помощью метода prompt() "Какую информацию хотите узнать о пользователе?".
Далее обращайтесь к свойствам объекта user через переменную info. Выведите результат с помощью alert.

Привет, ИМЯ!
